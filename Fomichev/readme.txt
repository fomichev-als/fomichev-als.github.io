Содержимое дизайн-пака


Все файлы в приобретенном дизайн-паке представлены в двух форматах:
готовом к использованию — PNG
и профессиональном — SVG (это векторный файл, позволяющий редактировать изображения без ограничений и делать распечатки любого размера).

1. Исходник логотипа: logo.png / logo.svg
2. Палитра фирменных цветов: palette.png / palette.svg
3. Визитка:
	― аверс: business_card_front.png / business_card_front.svg
	― реверс: business_card_back.png / business_card_back.svg
4. Фирменный бланк: letterhead.png / letterhead.svg
5. Паттерн: pattern.svg / pattern.png
6. Аватар для соцсетей: profile.png / profile.svg
